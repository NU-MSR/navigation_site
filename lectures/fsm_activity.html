FSM Activity

You are designing a finite state machine for a robotic vacuum cleaner that must vacuum itself and return to its charging station when it is low on batteries. The robot starts at its charging station with a full charge. When a user presses a button, the robot will start cleaning, if the robot is more than 75% charged. If the robot is less than 25% charged, it will return to the charging station.

  1. What states does the robot have?
  2. What are the events?
  3. Draw a state-transition diagram for the robot
  4. Implement a ROS node that models the robot:
    • When the robot is charging, it prints "Charging" and the power level
    • When the robot is vacuuming it prints "Vacuuming" and the power level
    • When the robot is returning to charge it prints "Returning" and the power level
    • Messages are printed once every 1 second
    • The battery charges at a rate of 10% per second
    • The battery discharges at a rate of 10% per second when vacuuming and 5% when returning
    • The robot always makes it back to the charging station when the battery is at 10% remaining
    • The user calls a "start" service to cause the robot to start vacuuming, if it has enough charge
      • The service can be a std_srvs::Empty
