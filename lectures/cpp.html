Compilation

Unlike Python, C++ is a compiled language, meaning that all the source code must be converted into executable machine code prior to running it. The two primary compilers on Linux are The GNU C++ compiler (G++) and the Clang compiler (clang++). There are also other, proprietary compilers such as the Intel C++ compiler.

The compilation process consists of several steps. In informal language, the collection of these steps is referred to as "compilation"

  1. Preprocess: this step handles pre-processor directives and strips comments (e.g., // for a single-line comment and /* */ for a block
    • The #include"file.h" literally inserts the text of file.h into the current file
    • The #define creates a macro. In C #define is used often, but in C++ there are usually better alternatives (especially with C++20
    • For the GNU compiler, the preprocessing is performed by cpp (which stands for C pre-processor)
  2. Compiling: The preprocessed files are converted to assembly language (human-readable low-level machine instructions)
    • For the GNU compiler, the compilation is performed by c++)
  3. Assembly: the assembly code is converted to binary machine code, in the form of object files
    • At the end of the day, after compilation and assembly, each .cpp file is converted to a single .o file
    • The .o files have executable code but cannot be run as-is because the data and code may refer to symbols in other files and the memory addresses of the code may not be assigned
    • For the GNU compiler, this step is handled by a program called gas (GNU assembler)
  4. Linking: This step takes the object files, and resolves all the symbols and addresses, forming the final executable
    • On Linux, the executable is in the form of an elf file (an executable file format).
    • For the GNU compiler, the linker is called ld.
  5. It is also possible to combine .o files into either static (compile time) or dynamic (run time) libraries. These are collections of compiled code that can be used by other code.

The g++ command (for the GNU compiler) automatically puts the C++ code through all these steps to generate
