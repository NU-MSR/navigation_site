Mobile Robot Kinematics

Lecture on Kinematics: See diffdrivekinematics.tex but use a different example. Define reference frames:

Each rigid body gets a frame. Each wheel gets a frame. There is a world frame.

Some conventions:

  1. The forward direction of the vehicle is the x axis of that vehicle's frame.
  2. Where applicable, it is easiest to have the y axis of the vehicle's frame aligned with at least one wheel axis.
  3. The axis of a wheel is the y axis and the x axis is the primary rolling direction.

Next, define the transforms between the frames: The variables from the world frame to the body frame become the state of the robot.

Next, write the twist at each wheel. For conventional wheels, the twist is angular velocity is the same as the angular velocity of the body (since the wheel is point contact, it can rotate in place. x velocity is based on the wheel spinning without slipping y velocity is 0 no slipping to the side.

Using adjoints, write the wheel twists in the body frame and set the equations equal to each other These are your kinematic equations, that should be solved

Digression

Deriving the adjoint in 2D:

Rotation in 2d is always about positive z axis, this means that in any frame the angular component of the rotation is the same. Activity to see this: two pieces of paper, draw several lines of different directions, match them up on both pages. Stack the paper on top of each other. Then, rotate the top paper about any point. Notice that all lines have rotated from there original ones by the same angle. Do the rotation again, by the same amount but by a different point. Notice all lines have rotated by same angle. Note however, the displacement is different.

FOr linear velocity: If you translate the sheet of paper X distance in T time, notice that all points move by the same distance, hence this linear velocity is a component. Note that if the coordinate system at the target point is rotated then thse velocities should be rotated as well.

Angular velocity on linear effect: Rotate the paper about a point. Notice that a point in the positive y direction moves in the -x direction, proportional to how high above it is

Notice that a point in the positive x direction moves in the y direction, proportional to how far it is.
